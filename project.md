프로젝트 이름 : ToDoT
프로젝트 실행 환경 : 윈도우 응용 프로그램
프로젝트 실행 조건 : 윈도우 7 이상
프로젝트 결과물 : 사용 가이드 문서, 윈도우에서의 빌드 방법 혹은 스크립트, exe 파일
프로젝트 개발 시 규칙 : 항상 수정 사항이 생기면 관련된 부분의 문서를 수정할 것.

프로젝트 기본 기능
- 원본 이미지/동영상을 도트 이미지/동영상 변환
- 원본 이미지/동영상을 아스키 코드 이미지/동영상 변환
- 원본 이미지/동영상을 한글 문자 이미지/동영상 변환
- 원본 이미지/동영상을 유니코드 이미지/동영상 변환

프로젝트 상세 설명
- 윈도우에서 동작할 수 있는 GUI 프로그램
- 프로젝트 실행 파일의 동일 레벨의 폴더로 이미지나 동영상 임시 tmp 디렉토리도 구현되어야 함. (업로드 실패 또는 작업 중인 파일들)
- 프로젝트 결과 파일이 저장될 동일 레벨의 폴더로 구현되어야 함.
- 최근 작업 진행한 히스토리도 10개 까지 볼 수 있도록 구현 되어야 함. 히스토리는 그냥 파일 탭 안에 리스트로 넣어줘.
- 기본 결과물 저장 폴더의 내용을 왼쪽에 미리보기와 리스트로 넣어줘.
- 결과물 저장 폴더의 내용을 자세히 보기, 썸네일만 보기, 썸네일+이름 보기 등 필요한 보는 방법도 변경할 수 있도록 해줘.
- 왠만한 이미지와 동영상 확장자 전부 지원 가능해야 함.
- 원본과 결과물 볼 수 있는 UI도 있어야 함.
- 이미지/동영상 원본 이미지와 동일한 크기로 결과물 UI 창에 띄워주되, 현재 윈도우 창 크기에 맞게 유동적으로 보여져야 함.
- New File UI : 내가 그림판 처럼 그릴 수 있도록 지원해줘. 이걸 변환 버튼 누르면 도트로 변할 수 있도록.
- Open File UI : 내 PC의 이미지, 동영상 선택할 수 있는 탐색기
- 픽셀 사이즈나 픽셀 지원 색상을 변경하면 미리보기로 결과물을 바로바로 볼 수 있도록 해줘.
- 프로그램 아이콘도 만들어서 입혀줘. ToDoT 를 의미할 수 있는 아이콘 만들어야 돼.
- 동영상 변환 시 동영상 미리보기도 재생해줘.
- 프로그램 설정 UI도 필요함
  - 설정 1 : 이미지, 동영상 결과물 지정
  - 설정 2 : 테마 변경(테마 최소 5개 이상)
- 그 외 필요하다고 생각하는 항목들도 추가해줘.
- 전반적인 UI는 시중에 유사한 프로그램들처럼 만들어줘.
- 동영상인 경우, 5초 가량 정도만 미리보기로 만들어서 바로 볼 수 있도록 해줘.
- 변환할 때 로딩 바에 예상 시간과 현재까지 진행된 시간도 표시해줘.
- 윈도우 작업 표시줄에도 로딩 볼 수 있도록 해줘.
- 이 프로그램을 CLI로도 지원할 수 있도록 해줘. 일단 미리보기나 이런건 안되곘지만, 기본적으로 우리가 수행하는 파라미터들, 파일명, 파일 output 위치등 지정할 수 있도록 만들어줘.
